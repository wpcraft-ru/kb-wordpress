# Beginner WordPress Designer

**Source:** https://learn.wordpress.org/course/beginner-wordpress-designer/
**Published:** 27 June 2024
**Last modified:** 31 March 2025
**Estimated duration:** 4 hours
**Enrolled learners:** 3732
**Average final grade:** 94%

---

This learning pathway equips designers new to WordPress with the skills to create professional WordPress websites. You'll learn to build a style guide, understand block themes, and design with a block-first approach. Explore best practices in web design, customize themes, master layout composition, and work with media blocks.

Some lessons in this learning pathway are duplicates of those in the user learning pathway.

## Course Structure

### 1. Introduction to designing with WordPress
- WordPress showcase for designers
- Using the Style Book as a design guide
- Overview of WordPress block theme terms and hierarchy
- Designing with a block-first mindset

### 2. Getting to know WordPress
- Finding a block theme for your design needs
- Choosing and installing a plugin
- Getting started with the WordPress dashboard
- Understanding the difference between WordPress posts and pages

### 3. Web design best practices
- Why design in the Site Editor and use it for prototyping?
- Web accessibility guidelines to create inclusive websites
- Testing your content for accessibility

### 4. Customizing themes and templates
- Intro to the Site Editor
- Differentiating between homepage display settings and various templates
- Using page templates
- Using template parts
- Customizing theme settings: Colors, fonts, typography and layout

### 5. Layout composition
- Designing with the Group block
- Designing with Row and Stack blocks
- Designing with the Columns block
- Dimensions: Padding, margin and block spacing

### 6. Working with media blocks
- Utilize the Gallery block
- Using the WordPress Media and Text block
- Uncovering the Cover block
- Using a plugin to create a carousel slider
