## Transcript

### Introduction
Plugins extend WordPress functionality: social media, analytics, security, marketing, membership, performance optimization, SEO, maps, contact forms, multilingual, e-commerce, caching. All achievable with the right plugins.

### Plugins Directory
Visit WordPress.org → Plugins Directory. All mentioned plugins are free in the repository; premium versions may offer more functionality at cost.

### Adding a plugin
Dashboard → Plugins → Add New. Shows featured, most popular, recommended, and favorites tabs. Use search function to find specific plugins.

### Evaluating a plugin — 4 criteria

1. **Star rating** — feedback from users who installed the plugin
2. **Last updated** — indicates active support. Avoid plugins not updated in a long time
3. **Active installations** — indicates popular, well-supported plugin
4. **Compatibility** — must be compatible with latest WordPress version

Click "More details" to see support level, FAQ, and reviews.

### Installing and activating
Example: Search for forms plugin (Contact Form 7, WP Forms, Ninja Forms). Install → Activate → Set up → Add form block to any page.

You can also upload a zip archive of a plugin from local computer (useful for premium plugins downloaded outside the Directory).

### Deactivating and deleting
Go to Plugins section → Deactivate → Delete permanently.

### Troubleshooting plugin conflicts
**Two methods:**
1. Manually deactivate plugins, test individually to find the culprit
2. Use a troubleshooting plugin (e.g. Health Check and Troubleshooting)

Remember: install well-built plugins using the 4 criteria.
