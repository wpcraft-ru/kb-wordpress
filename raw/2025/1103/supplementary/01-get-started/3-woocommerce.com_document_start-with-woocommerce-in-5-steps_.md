Otherwise, you can download from[WordPress.org](https://wordpress.org/download/)and then install by following instructions at [How to install WordPress](https://wordpress.org/support/article/how-to-install-wordpress/).

### Activate WooCommerce

With your self-hosted website up and running, it’s time to install ecommerce functionality with **WooCommerce**. There are three ways to do this.

*   [**Create a free account**](https://woocommerce.com/sso?new=1&next=%2Fwoocommerce%2F)on WooCommerce.com 
    *   Sign-up for an account on WordPress.com and WooCommerce.com
    *   Select from: 
        *   A pre-installed solution with a trusted host
        *   Auto-installing WooCommerce if you have a WordPress site already
        *   Download the zip file and do it yourself 

*   Install WooCommerce 
    *   [**Download**](https://wordpress.org/plugins/woocommerce/) the WooCommerce plugin 
        *   Download the plugin
        *   Go to your website
        *   Go to: Dashboard > Plugins > Add New > Upload
        *   Select the downloaded plugin
        *   Install
        *   Activate 

    *   **Search** for the WooCommerce plugin from within your website 
        *   Go to your website
        *   Go to:**Dashboard > Plugins > Add New > Search Plugins**
        *   Search for “WooCommerce”
        *   Install
        *   Activate 

Upon activating the plugin, you are prompted with a [Setup Wizard](https://woocommerce.com/document/woocommerce-setup-wizard/). We strongly advise following these steps, as it takes you through initial setup.

### [Pick a Theme](https://woocommerce.com/document/start-with-woocommerce-in-5-steps/#pick-a-theme)

[↑ Back to top](https://woocommerce.com/document/start-with-woocommerce-in-5-steps/#doc-title "Back to top")
In the **Setup Wizard** mentioned above, you will be asked to pick a theme. The theme you choose will determine the starting look and layout of your store/site (though you can customize it from there!). You can use a free theme included with WordPress or purchase a premium theme.

We recommend **[Storefront](https://wordpress.org/themes/storefront)**and its**[child themes](https://woocommerce.com/product-category/themes/storefront-child-theme-themes/)**because they are built and optimized for WooCommerce. Storefront is free. If you wish to add functionality, you can purchase**[Storefront extensions](https://woocommerce.com/product-category/storefront-extensions/)**.

If Storefront isn’t for you, there are lots of other **[themes](https://woocommerce.com/themes/)** that work great with WooCommerce.

### [Extend WooCommerce](https://woocommerce.com/document/start-with-woocommerce-in-5-steps/#extend-woocommerce)

[↑ Back to top](https://woocommerce.com/document/start-with-woocommerce-in-5-steps/#doc-title "Back to top")
Grow your store by adding functionality with**[extensions](https://woocommerce.com/products)**. We sell extensions for nearly everything: [Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/), [Bookings](https://woocommerce.com/products/woocommerce-bookings/), [Memberships](https://woocommerce.com/products/woocommerce-memberships/), [payment gateways](https://woocommerce.com/product-category/woocommerce-extensions/payment-gateways), [shipping](https://woocommerce.com/product-category/woocommerce-extensions/shipping-methods),and much more.

You can start a store/website for free and scale up as your customer base and revenue grow.Each extension is purchased on a subscription that gives you access to support and product updates. A subscription is billed annually.
