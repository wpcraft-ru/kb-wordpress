# Customizing the Product Catalog Template

The Product Catalog template controls the layout and design of your store’s main Shop page. By default, this page displays all published products in your store. Using the **[Site Editor](https://woocommerce.com/document/woocommerce-store-editing/the-editor/#access-the-editor)**, you can customize the Product Catalog template to reflect your brand, highlight key products, and create a shopping experience tailored to your customers.

This template is powered by the **[Product Collection block](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/)**, which dynamically displays products. You can rearrange, style, or extend it with additional blocks, patterns, and filters.

## [Access Product Catalog Template](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#access-product-catalog-template)

[↑ Back to top](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#doc-title "Back to top")
1.   From your WordPress dashboard, go to **Appearance > Editor**
2.   Select **Templates >****WooCommerce**
3.   Choose **Product Catalog**

![Image 2](https://woocommerce.com/wp-content/uploads/2025/09/image_016ce0.png?w=980)

**Note:** The Product Catalog template takes priority over the Shop page. If you’ve made edits directly to your Shop page and they aren’t appearing on the frontend, it’s likely because **this template is overriding them**. To customize your shop’s appearance, edit the Product Catalog template instead.

## [Powered by the Product Collection Block](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#powered-by-the-product-collection-block)

[↑ Back to top](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#doc-title "Back to top")
The Product Catalog template is powered by the **[Product Collection block](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/)**, which displays your products in a customizable grid or list.

By default, this block shows all published products, but you can adjust its settings to control layout, styling, and product display to fit your store’s needs. You can learn more about the Product Collection Block in the guides below:

*   [Product Collection Block Structure](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/#block-structure)
*   [Product Collection Block Settings](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/#section-13)
*   [Display and Layout Options](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/#section-15)
*   [Query Settings](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/#section-14)

## [Inner Blocks of the Product Catalog Template](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#inner-blocks-of-the-product-catalog-template)

[↑ Back to top](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#doc-title "Back to top")
By default, the Product Catalog template includes several inner blocks that work together to create the Shop layout.

| **Block** | **Purpose** |
| --- | --- |
| **Store Breadcrumbs** | Displays a navigation trail for the current page. |
| **Archive Title** | Shows the name of the current archive. |
| **Term Description** | Display the description of categories, tags, and custom taxonomies when viewing a catalog. |
| **Store Notices** | Display shopper-facing notifications (e.g., “Product added to cart”) from WooCommerce or extensions. |
| **Product Results Count** | Shows the number of products from the catalog or search result page. |
| **Catalog Sorting** | Allows customers to reorder products by various criteria. |
| **[Product Collection](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/)** | Displays a grid of products in the selected category. |

## [Adding Product Filters](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#adding-product-filters)

[↑ Back to top](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/#doc-title "Back to top")
To improve navigation and help shoppers find products faster, add the **[Product Filters block](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/filter-products/)** to your Product Catalog template. Customers can refine products by:

*   Price range
*   Product Rating
*   Attributes
*   Stock Status
*   Category
*   Brand 
*   Tag
