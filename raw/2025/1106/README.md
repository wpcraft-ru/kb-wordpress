## Краткие выводы

### 🏷️ Типы плагинов

| Тип | Плагины |
|-----|---------|
| **Классические SEO** | Yoast, Rank Math, AIOSEO, SEOPress, The SEO Framework |
| **Панель интеграции** | Site Kit for Yandex (аналог Site Kit by Google) |

> Site Kit for Yandex — комплементарный инструмент. Он дополняет SEO-плагин данными из экосистемы Яндекса, а не заменяет его. Можно использовать вместе с любым SEO-плагином.

### 🏆 Лидеры по категориям

| Категория | Победитель |
|-----------|-----------|
| Бесплатная функциональность | **Rank Math** |
| Для новичков | **Yoast SEO** |
| WooCommerce / Бизнес | **AIOSEO** |
| ИИ-генерация мета-тегов | **SEOPress** |
| Скорость / Минимализм | **The SEO Framework** |
| Яндекс.Вебмастер | **Rank Math** / **SEOPress** |
| Цена для агентств | **SEOPress** ($149 unlimited) |

### 📊 Ключевые цифры

- Yoast SEO в 4 раза медленнее Rank Math по влиянию на загрузку
- The SEO Framework потребляет всего 23.71 KB памяти
- Rank Math даёт 5 ключевых слов бесплатно, Yoast — 1
- SEOPress не ограничивает ключевые слова для анализа
