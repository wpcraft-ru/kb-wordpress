## GitHub
- 100M+ разработчиков
- GitHub Actions: 2 000 бесплатных минут/мес
- GHCR (Container Registry) бесплатно
- Copilot: $10/мес
- Security scanning (advanced — платно)
- Self-hosted: только Enterprise (платно)
- GitHub Team: $4/пользователь/мес, 3 000 CI минут
- Принадлежит Microsoft

## GitLab
- DevSecOps all-in-one
- 400 бесплатных CI минут/мес
- Встроенный Container Registry (бесплатно)
- Security scanning (базовый — бесплатно)
- GitLab Duo (ИИ-ассистент)
- Self-hosted: CE бесплатно
- GitLab Premium: $29/пользователь/мес, 10 000 минут
- Best for: enterprise, compliance, сложные пайплайны

## Bitbucket
- Интеграция с Jira/Confluence/Trello
- 50 бесплатных CI минут/мес
- Нет нативного Container Registry
- Security через интеграции
- Self-hosted: Data Center (платно)
- Bitbucket Standard: $3/пользователь/мес
- Best for: Atlassian-команды

## Сравнение
| Feature | GitHub | GitLab | Bitbucket |
|---------|--------|--------|-----------|
| Free CI/мес | 2 000 мин | 400 мин | 50 мин |
| Self-hosted | Enterprise ($) | CE (free) | Data Center ($) |
| Container Registry | GHCR (free) | Built-in (free) | Нет |
| Security scanning | Advanced ($) | Basic (free) | Через интеграции |
| AI | Copilot ($10/мес) | Duo | Atlassian Intelligence |
| Jira | Хорошо | OK | Нативно |
